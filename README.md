# TMX Platform Full-Pack v1.1 (Auth + Dashboard + Blue UI)

- Frontend com UI atualizada (paleta azul), Login/Registro com UX melhor, Dashboard protegido.
- Backend com JWT+bcrypt, rotas TMX protegidas.
- Testes prontos (REST Client .http) e sample TMX.

## Rodando
1. Backend
   ```bash
   cd backend
   npm install
   npm run dev
   ```
2. Frontend
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
